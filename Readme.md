# AI Driver Drowsiness Detection with Wheel Motion Control

This project detects driver fatigue using computer vision and controls wheel motion using an ESP based hardware system.

## System Components

AI Detection
- Eye detection using OpenCV
- Drowsiness classification

Hardware Control
- ESP microcontroller
- Wheel motion monitoring
- Alert system

## Tech Stack
Python
OpenCV
ESP32 / Arduino
Embedded C
Computer Vision
